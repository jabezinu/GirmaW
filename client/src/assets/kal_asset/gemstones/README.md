# Gemstone Images

Place gemstone images in this folder with the following filenames:

## Required Images:

1. `blue-sapphire.jpg` - Blue Sapphire from Ceylon
2. `ruby.jpg` - Ruby from Burma
3. `emerald.jpg` - Emerald from Colombia
4. `pink-tourmaline.jpg` - Pink Tourmaline from Brazil
5. `aquamarine.jpg` - Aquamarine from Pakistan
6. `amethyst.jpg` - Amethyst from Uruguay
7. `yellow-sapphire.jpg` - Yellow Sapphire from Ceylon
8. `tanzanite.jpg` - Tanzanite from Tanzania
9. `garnet.jpg` - Garnet from Madagascar
10. `citrine.jpg` - Citrine from Brazil
11. `blue-topaz.jpg` - Blue Topaz from Brazil
12. `peridot.jpg` - Peridot from Pakistan

## Image Specifications:
- Format: JPG or PNG
- Recommended size: 600x600px or larger
- Aspect ratio: Square (1:1) preferred
- Quality: High resolution for best display

## Note:
The images are referenced in `/src/data/gemstonesData.js`. If you change filenames, update the data file accordingly.
